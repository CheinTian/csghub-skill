# openai.CompletionUsageCompletionTokensDetails

**Type:** object

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `accepted_prediction_tokens` | integer | No | When using Predicted Outputs, the number of tokens in the prediction that
appeared in the completion. |
| `audio_tokens` | integer | No | Audio input tokens generated by the model. |
| `reasoning_tokens` | integer | No | Tokens generated by the model for reasoning. |
| `rejected_prediction_tokens` | integer | No | When using Predicted Outputs, the number of tokens in the prediction that did
not appear in the completion. However, like reasoning tokens, these tokens are
still counted in the total completion tokens for purposes of billing, output,
and context window limits. |
